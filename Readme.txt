Venta de Libros utilizando Aeternity Blockchain
-----------------------------------------------

La aplicación permite ingresar nuevos libros para la venta, con un importe a utilizar como referencia al momento de la compra.
Está pensada para que la compre un interesado y se contacte con el email del vendedor.
El objetivo fue probar la plataforma y por lo tanto no tiene todas las validaciones necesarias para funcionar en producción.
Es una adaptación del código del curso, donde se puede interactuar con blockchain.

Al ingresar un libro para la venta, se genera un elemento con estado "A LA VENTA"

Para comprar un libros se debe ingresar el número a comprar y presionar el botón "Comprar Libro por Nro".
